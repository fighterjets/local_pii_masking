EDGE CASES
==========

Should Detect:
--------------
Valid UEN: 201234567A ✅
Valid USCC: 91110000000000000A ✅
Valid BR: 12345678 ✅
Valid BR with suffix: 12345678-000 ✅

Should NOT Detect (False Positives):
-------------------------------------
Short number: 12345 ❌
Random 8 digits in date: 20231215 (might match BR, acceptable false positive)
Phone number: 91234567 (might match BR segment, but context different)
Random 18 chars: ABCDEFGHIJ12345678 (might match USCC if all valid chars)

Note: Some false positives are acceptable (better to detect than miss).
Corporate registration numbers are less distinct than personal IDs.
Context-based filtering can be added later if needed.
